Résponsabilité

De nos jours, les sites web ont besoin de faire preuve d'une certaine responsabilité. La quantité d'informations disponible à travers notre usage du web est astronomique.

Les cookies permettent aux "trackers" de nous identifier. Attention, cela ne signifie pas qu'ils aient notre adresse personnelle ou notre numéro de téléphone ! Cela signifie que d'une session de navigation à une autre, un site web peut identifier un revenant.

Dès que votre navigateur tente d'afficher une publicité tierce, il se présente à un serveur. Ce serveur peut demander l'usage de cookies, vous voyez où on en arrive ? Même si votre navigation se fait sur de divers sites, les entreprises de publicités ne sont pas si nombreuses et peuvent appercevoir votre navigation. On parlera même d'un monopole pour certains. Les publicités n'ont rien de mauvais, ni même les cookies, c'est leur usage qui n'est pas fait de manière respectueuse.

Lorsque votre navigateur lance une requête, il donnera toujours une information au serveur distant : le "referrer". Cela correspond tout simplement à l'addresse du site principal sur lequel vous êtes. Si vous êtes sur "https://example.com/nuitdelinfo/", toute requête envers un serveur de publicités recevra cette URL et saura donc plus ou moins ce que vous faisez.

Donnons un exemple pratique : vous utilisez un site we A et un site web B. Les deux se servent de publicités afin de se faire un peu d'argent et de financer leur usage gratuit. Jusqu'ici, il ne se passe rien de mal. Ces sites web peuvent utiliser des cookies pour stocker des réglages : peut-être que vous préférez un thème d'affichage nuit et vous ne voulez pas faire ce choix à chaque fois que vous chargez une page.

Le problème arrive lorsque l'on fait le choix de publicités à afficher. Le choix le plus simple en général serait d'aller voir une entreprise C qui se charge entièrement de récolter les publicités à afficher. Comme ça, chacun son boulot : ils le font sûrement mieux que A et B ne le feraient s'ils étaient seuls. Il n'y a pas énormément de choix d'entreprises (on parle toujours d'un monopôle).

Lorsque vous recevez les publicités sur le site A, l'entreprise C sait que vous étiez sur le site A. Maintenant, ils pouront vous afficher des publicités sur le site B qui ont un rapport avec le contenu du site A.

Il est donc très important de comprendre le mécanisme des cookies, de savoir pourquoi ils existent. À l'origine, ils ne servaient que de stockage temporaire pour vos réglages, ou encore pour que le serveur se rappelle que vous êtes connectés avec votre identifiant et mot de passe lors d'une session sur le site web de votre banque par exemple.

Avec ces informations, nous espérons que vous pourrez faire des choix pertinents sur la collecte de vos données personnelles et que vous compreniez mieux les mécanismes utilisés pour vous identifier sur le web.

Nous accordons une importance inestimable à votre vie privée
Afin de pouvoir avoir un usage correct de notre page web, il vous faudra accepter quelques cookies fonctionnels essentiels. Pour ce faire, nous avons besoin de récolter tout ce dont vous disposez. Seront inclus dans nos collectes : vos clics, vos soupirs, vos biscuits préférés, la durée de vos poses-thé.
