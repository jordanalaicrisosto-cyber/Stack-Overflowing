# Stack-Overflowing

Repository crée spécialement pour la nuit de l'info, dans le but de réaliser divers défis durant celles-ci.

## Comment utiliser ?

Il suffit de cloner le dépôt et ensuite d'éxécuter `python3 build.py` pour générer un dossier `output/` contenant la page statique finale.

On pourra ensuite démarrer un serveur local avec la commande `python3 -m http.server --directory output/` ou avec `npx serve output/` si on a NodeJS d'installé.
